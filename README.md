# Linux System Administration Labs

This repository contains multiple hands-on Linux system administration projects built using Ubuntu and Red Hat Enterprise Linux.

## Projects Included

- Web Server Setup (Apache on Ubuntu)
- LVM Storage Management
- Bash Automation Scripts
- SSH Security Hardening

## Purpose

The goal of these labs is to demonstrate practical experience in Linux system administration, networking, storage, and automation.

## Tools Used

- Ubuntu Server
- Red Hat Enterprise Linux
- Apache2
- Bash Shell
- LVM
- SSH

## Note

Each folder contains project-specific notes and configurations.
